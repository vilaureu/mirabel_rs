# _mirabel_rs_

_mirabel_rs_ provides a Rust wrapper for writing
[_mirabel_](https://github.com/RememberOfLife/mirabel) plugins.

An example plugin can be found in the `example` folder.

## Implemented wrappers

- [x] Frontend
- [ ] Game Wrap
- [ ] Engine Wrap
